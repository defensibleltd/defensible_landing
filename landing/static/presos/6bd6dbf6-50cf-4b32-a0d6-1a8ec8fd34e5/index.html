HPE Aruba Private 5G Solution Overview

HPE Aruba Networking Private 5G is a comprehensive enterprise cellular solution built on proven technology from Athonet, which HPE acquired in 2023. The solution provides an integrated, end-to-end private 5G network designed to "complement existing Wi-Fi infrastructure rather than replace it".

Main Feature Benefits

  • Comprehensive coverage and mobility: Wider coverage per radio, seamless indoor/outdoor coverage, improved network-controlled handoff and high mobility support
  • Advanced Quality of Service (QoS): Fine-grained, deterministic QoS and SLAs for latency-sensitive, mission-critical traffic
  • Minimal wireless interference: Spectrum is protected and managed (e.g., via SAS for CBRS in the US)
  • Security: SIM-based device identity, separation from public networks

What's in the "stack"?

The solution requires different devices and controllers compared to traditional Wi-Fi deployments

  • Small Cells: HPE-manufactured 4G/5G small cell radios for wider indoor and outdoor coverage (better propagation)
  • Core Infrastructure: HPE ProLiant Gen11 servers running the mobile core software
  • Management: Dedicated private 5G cloud-based management dashboard (separate from Aruba Central initially) plus configuration wizards designed to mask 3GPP cellular complexity
  • Client Devices: 5G-capable devices with SIM/eSIM cards for cellular connectivity, spectrum access system (for CBRS in the US)

Integration with Existing Networking

HPE's approach focuses on unified wireless connectivity rather than siloed deployment. Currently, private 5G management operates through a separate dashboard from HPE Aruba Central, though both systems are cloud-based with similar interfaces

Note: the solution fully supports CBRS (Citizens Broadband Radio Service) spectrum in the USA. The HPE solution includes a Spectrum Access System (SAS) to manage CBRS spectrum coordination and interference protection. This eliminates the need for enterprises to acquire expensive licensed spectrum from mobile operators

Future Integration Plans

HPE is working toward full integration with Aruba Central, which will provide unified management across wired, Wi-Fi, private 5G, and SD-WAN networks under a single pane of glass. This integration will include:

  • Shared identity management using (e)SIM credentials as primary identity
  • Unified quality of service policies
  • Common security frameworks and analytics tools
  • Wi-Fi PassPoint technology for seamless device onboarding between networks

Future Integration with Wi-Fi

HPE is developing Air Pass technology to bridge this gap by enabling devices with SIM credentials to automatically authenticate on Wi-Fi networks. However, Air Pass currently only works to allow cellular-credentialed devices to access Wi-Fi, it does not enable Wi-Fi devices to access private 5G.

WLAN Pros Community Feedback on P5G

Main actual viable sectors/use cases:

  • Healthcare
  • Warehousing/logistics/robotics/autonomous vehicles
  • Outdoor/underground mining/industrial
  • Energy sector
  • Some back of house/front of house LPV(Large Public Venue)

Challenges for Enterprise:

  • Brownfields (leverage WLAN unless specific requirements)
  • Pricepoint (additional deployment unless greenfields)
  • Some scenarios necessitate neutral host 5G
  • MDM(Mobile Device Management) for SIM/eSIM management/policies
